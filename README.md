### Package installation

```bash
npm install

```

### Building

To create a production version of your app:

```bash
npm run build
```

### Local host deployment

```bash
npm run dev

```

### Cloud flare deployment

Configure wrangler:

```bash
wrangler login

```

Deployment:

```bash

npm run deploy
```
